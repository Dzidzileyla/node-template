## Development setup

1. Clone repository:

```sh
git clone git@github.com:Dzidzileyla/node-template.git
```

2. Run docker configuration for development stage:

```sh
docker-compose -f docker-compose-dev.yml up
```

3. Using postman check 0.0.0.0:8081.

4. Ready to go!
